# ECOM-800

　トランジスタ技術 1977年10月号から1978年1月号にかけて連載されていた「マイコンピュータをつくろう(SC/MP編)」のECOM-800の基板を起こしました。

## 回路図
　KiCadフォルダ内のECOM-800.pdfを参照してください。

　掲載されている回路から以下の変更をしています。

　・ワンボードとするためにロータリースイッチをスライドスイッチに変更しています。

　・81LS95を74LS244に変更しています。

　・SRAMを6264としています。

　・SRAMのバッテリーバックアップを付加しました。

## 組み立ての注意点
　SW1の取付向きに注意してください。押されていない状態で上側がONとなるようにつけます。

## バッテリーバックアップ
　J2のEXT側でCR2032でのバックアップになり、IN側で電源ラインでのバックアップになります。

　CR2032を使わなくても電源に電池等を使ってIN側しておけば電池を外さない限りバックアップされます。

　なお、ECOM-800の回路の特性として電源スイッチがONになった時にデータ・スイッチ(D0～D7)の状態がメモリ0000hに書き込まれてしまうようです。(アドレススイッチの状態に関係なく、0000hに書き込まれます)

　0000hに意図して書き込んだ内容が勝手に書き換わってしまいますが、SC/MPⅡで0000hは実行されませんので書き換わってしまっても影響はありません。
